# Qt-ICQ
项目描述：个人练习项目，QT实现多个客户端进行通讯

使用方法：先启动服务器，后启动客户端，可以启动多个客户端。客户端之间可以在公屏进行信息通讯，类似群聊。也可以两个客户端点对点的通讯，类似好友私聊。

涉及技术：QT基础控件，QT槽函数，QT自定义信号和槽，QT套接字通信，基于TCP/IP的网络编程，QT处理Json格式的数据，QListView控件的使用
